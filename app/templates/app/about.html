    About This Project

    CS 3240 - Fall 2021

    The application is created by 5 undergraduate students for CS 3240: Advanced Software Development Techniques in fall 2021 at the University of Virginia. It allowers users to search, filter, find, and leave comments/ratings about off-grounds housing for the university. This is a semester long project and the software uses Django, Python, Bootstrap, Heroku, and other API services. If you want to learn more about us, please refer to our personal links below.
